
Note: This app is meant to be a proof of concept.
## Overview

- Displays a map, a location icon, a scroll to select time, a search button

- Allows the following actions:

    * locating device on a map

    * view the pedestrian count on a street

    * search any place on map

    * look for historical pedestrian count using time scrolling selector  

 

## Build Requirements


Xcode 8.3 (iOS 10.3.3) or later

 

## Runtime Requirements

iOS 10.3.3
